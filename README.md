# Marketing-Funnel-Optimization-Through-A-B-Testing
An end-to-end statistical analysis of marketing campaign performance across engagement, purchase intent, and conversion efficiency.

## Objective

Evaluate whether the test marketing campaign performed differently from the control campaign using statistical A/B testing.

## Business Question

Did the test campaign significantly impact conversion performance compared to the control campaign?

## Methods

- Two-proportion z-test
- Confidence interval estimation
- Mann–Whitney U test
- Funnel metric analysis

## Metrics Analyzed

- Conversion rate
- Click-through rate (CTR)
- Add-to-cart rate
- Cost per purchase

## Key Findings

- Conversion rate difference was statistically significant
- CTR and add-to-cart behavior differed significantly
- Cost per purchase did not differ significantly
- Test campaign underperformed relative to control

- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
